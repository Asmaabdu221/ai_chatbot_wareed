## 🔄 طريقة تحديث التحاليل (للمستقبل)

### **الطريقة الأولى: تلقائياً عبر API** ⭐ (موصى بها)

**الخطوات:**
1. تجهيز ملف Excel (نفس تنسيق الملف الحالي)
2. رفع الملف عبر API (انظر الأمثلة أدناه)
3. التغييرات تصبح فعالة خلال 30 ثانية
4. النظام ينشئ نسخة احتياطية تلقائياً

**مثال (استخدام برنامج Postman أو cURL):**
```
POST http://localhost:8000/api/admin/upload-excel
Header: X-Admin-API-Key: مفتاحك-السري
Body: Form-data, file = ملف Excel
```

**النتيجة:** ✅ تحديثات فورية بدون إيقاف

---

### **الطريقة الثانية: يدوياً (احتياطية)**

**إذا لم يعمل API:**
1. إرسال ملف Excel للمطور
2. المطور يشغل السكريبت:
   ```bash
   python simple_excel_converter.py
   ```
3. إعادة تشغيل الخادم

**الوقت:** 5-10 دقائق

| **نوع العينة** | نوع العينة المطلوبة | "Serum" |
| التحاليل المكملة | تحاليل أخرى مفيدة | (اختياري) |
| تحاليل قريبة | تحاليل مشابهة | (اختياري) |
| تحاليل بديلة | تحاليل بديلة | (اختياري) |

## 🔐 الأمان

### كلمة المرور (API Key):
- يجب إضافة كلمة مرور سرية في ملف `.env`
- الإضافة:
  ```
  ADMIN_API_KEY=your-secret-password-here-123
  ```
- **لا تشاركي هذه الكلمة مع أحد**
- استخدميها في الـ Header عند رفع الملفات

## 🆘 ماذا أفعل إذا حدث خطأ؟

### المشكلة: رفعت ملف Excel ولكن البيانات غير صحيحة

**الحل:**
1. النظام أنشأ نسخة احتياطية تلقائياً
2. يمكن استعادة البيانات القديمة:
   ```bash
   GET http://localhost:8000/api/admin/backups
   # للحصول على قائمة النسخ الاحتياطية
   
   POST http://localhost:8000/api/admin/restore-backup/knowledge_backup_20260204_120000.json
   # لاستعادة نسخة معينة
   ```

**أو:**  
اتصلي بالمطور وسيقوم بإعادة النسخة الاحتياطية
