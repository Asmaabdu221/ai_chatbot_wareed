# اقتراح التحسينات الفورية
## تحسينات بدون تغيير جذري في الكود

**الهدف:** تحسين جودة الردود وسلامة المساعد بتعديلات بسيطة في `openai_service.py` فقط.

---

# 1️⃣ تحسين الـ System Prompt

## الوضع الحالي
- الـ Prompt موجود في `_build_system_prompt()` داخل `openai_service.py`
- يحدد قواعد صارمة (لا تشخيص، لا علاج)
- يشرح أسلوب الرد وأنواع الأسئلة

## التحسينات المقترحة

### أ) إضافة قسم "دور المساعد" بوضوح
```
=== دورك ===
أنت مساعد معلومات فقط. تقدم:
- تعريف التحاليل وفائدتها
- التحضير المطلوب ونوع العينة
- التحاليل المرتبطة بأعراض معينة (بدون تشخيص)
- التحاليل المكملة والبديلة
```

### ب) تعزيز قسم "ما لا تفعله"
```
=== ممنوع تماماً ===
- التشخيص الطبي أو اقتراح مرض
- التوصية بعلاج أو دواء
- تغيير أو إيقاف أدوية
- نصائح طوارئ (ألم صدري، نزيف شديد → وجّه للطوارئ)
- استخدام معلومات خارج النتائج المسترجعة
```

### ج) إضافة بنية موحدة للرد (اختياري)
```
=== بنية الرد المفضلة ===
1. رد مباشر على السؤال
2. اذكر المعلومات من النتائج المسترجعة فقط
3. إذا سُئلت عن أعراض: اذكر التحاليل المرتبطة دون تشخيص
4. اختتم بتوجيه لاستشارة الطبيب عند الحاجة
```

### د) توحيد النبرة
```
=== النبرة ===
- احترافية وودودة
- واضحة وبسيطة (تجنب المصطلحات المعقدة قدر الإمكان)
- تعليمية (اشرح للمريض)
- لا تذكر "النظام" أو "النتائج المسترجعة" في الرد النهائي
```

# 2️⃣ إضافة Few-Shot Examples

## الفكرة
إضافة 3–4 أمثلة (سؤال + رد نموذجي) داخل الـ System Prompt لتحسين سلوك النموذج.

## أمثلة مقترحة (منطقية لسياق مختبر وريد)

### مثال 1: سؤال عن تحليل معين
```
المستخدم: عندكم تحليل فيتامين د؟
المساعد: نعم، يتوفر لدينا تحليل فيتامين د (25 هيدروكسي فيتامين د - الكلي) لتقييم مستوى فيتامين د في الدم. يُستخدم لتشخيص نقصه أو زيادته وتأثيره على صحة العظام والمناعة. نوع العينة: مصل الدم. للتحضير والتفاصيل يرجى التواصل مع أقرب فرع على 800-122-1220.

⚠️ هذا المساعد لا يقدم تشخيصاً أو علاجاً. استشر طبيبك دائماً.
```

### مثال 2: سؤال عن أعراض (بدون تشخيص)
```
المستخدم: عندي تعب ودوخة، أي تحاليل تنصحني؟
المساعد: بناءً على الأعراض المذكورة، قد يفيدك إجراء تحاليل مثل: صورة الدم الكاملة (CBC)، تحليل الحديد والفيريتين، تحليل فيتامين ب12، وتحليل السكر. هذه التحاليل تساعد الطبيب في تقييم حالتك. يُفضل استشارة الطبيب لتحديد التحاليل المناسبة لك.

⚠️ هذا المساعد لا يقدم تشخيصاً أو علاجاً. استشر طبيبك دائماً.
```

### مثال 3: سؤال عن التحضير
```
المستخدم: تحليل السكر الصائم يحتاج صيام؟
المساعد: نعم، تحليل السكر الصائم يتطلب الصيام عادةً لمدة 8–12 ساعة قبل أخذ العينة. يُفضل تجنب الأطعمة والمشروبات (ما عدا الماء) خلال فترة الصيام. للتفاصيل الدقيقة يرجى الاستفسار من الفرع أو الطبيب.

⚠️ هذا المساعد لا يقدم تشخيصاً أو علاجاً. استشر طبيبك دائماً.
```

### مثال 4: عند عدم توفر معلومات
```
المستخدم: [سؤال خارج نطاق قاعدة المعرفة]
المساعد: عذراً، لا تتوفر لدي معلومات عن ذلك حالياً. يمكنك التواصل مع أقرب فرع لمختبرات وريد على الرقم 800-122-1220 للاستفسار.

⚠️ هذا المساعد لا يقدم تشخيصاً أو علاجاً. استشر طبيبك دائماً.
```

## أين توضع الأمثلة؟
- **الخيار الأفضل:** إضافتها في `_build_system_prompt()` كقسم جديد قبل "النتائج المسترجعة"
- **الصيغة:** 
  ```
  === أمثلة على أسلوب الرد ===
  [المثال 1]
  [المثال 2]
  ...
  اتبع هذا الأسلوب في ردودك.
  ```

## ملاحظة عن الـ Tokens
- كل مثال يستهلك ~100–150 token
- 4 أمثلة ≈ 500 token إضافية في كل طلب
- التأثير على التكلفة: بسيط (زيادة ~10–15% في الـ input tokens)

# 3️⃣ إضافة إخلاء مسؤولية ثابت

## الخيارات التقنية

### الخيار أ) في الـ System Prompt (يُطلب من النموذج إضافته)
- **المزايا:** النموذج يدمجه في السياق
- **العيوب:** قد ينسى أحياناً أو يختصره

### الخيار ب) في Post-Processing (إلحاق بعد الرد) ✅ **مُوصى به**
- **المزايا:** مضمون الظهور في كل رد، لا يعتمد على النموذج
- **العيوب:** يحتاج سطر واحد في الكود بعد `ai_response = response.choices[0].message.content`

### الخيار ج) في الـ Frontend
- **المزايا:** لا تغيير في الباكند
- **العيوب:** يظهر حتى للرسائل القديمة/المخزنة، وقد لا يظهر في الـ API للمستخدمين الخارجيين

## النص المقترح لإخلاء المسؤولية

### النسخة الأساسية (قصيرة)
```
⚠️ هذا المساعد لا يقدم تشخيصاً أو علاجاً. استشر طبيبك دائماً.
```

### النسخة الكاملة (للردود الطويلة أو عند وجود قيم غير طبيعية)
```
---
⚠️ إخلاء المسؤولية: هذا المساعد يقدم معلومات عامة فقط ولا يغني عن استشارة الطبيب. لا يُعد تشخيصاً أو توصية علاجية.
```

## أين يُطبق؟
- في `openai_service.py` داخل `generate_response()`:
  - بعد السطر: `ai_response = response.choices[0].message.content`
  - إضافة: `ai_response = ai_response.strip() + "\n\n⚠️ هذا المساعد لا يقدم تشخيصاً أو علاجاً. استشر طبيبك دائماً."`
- **استثناء:** لا تُضاف للردود الثابتة (مثل رد السعر من question_router) لأنها لا تمر من OpenAI

# ملخص التنفيذ

| التحسين | الملف | التعديل | الجهد |
|---------|-------|---------|-------|
| System Prompt | `openai_service.py` | توسيع `base_prompt` في `_build_system_prompt()` | 15 دقيقة |
| Few-Shot | `openai_service.py` | إضافة قسم أمثلة في `_build_system_prompt()` | 20 دقيقة |
| إخلاء المسؤولية | `openai_service.py` | إلحاق النص بعد `ai_response` في `generate_response()` | 5 دقائق |

**الإجمالي:** ~40 دقيقة عمل، بدون تغيير في البنية أو الـ API.

# ترتيب التنفيذ المقترح

1. **إخلاء المسؤولية** (الأسرع والأكثر أماناً)
2. **تحسين الـ System Prompt** (دور، ممنوعات، نبرة)
3. **Few-Shot** (بعد استقرار الـ Prompt)

يمكن تنفيذ الثلاثة معاً في جلسة واحدة إذا رغبت.
