## 📊 تقدير الاستهلاك

### قبل التحسين:
- **متوسط التوكنز:** ~1,500 توكن/طلب
- **التكلفة:** ~$0.0015/طلب
- **عدد الطلبات بـ $5:** ~3,300 طلب

### بعد التحسين:
- **متوسط التوكنز:** ~500-700 توكن/طلب ✅
- **التكلفة:** ~$0.0005/طلب ✅
- **عدد الطلبات بـ $5:** ~10,000 طلب ✅

**التوفير:** 70% من التكلفة!

## 💵 تفاصيل تسعير gpt-3.5-turbo

|------|---------------------|

| Output (الإخراج) | $0.0015 |

**مثال:**
- 500 توكن إدخال = $0.00025

## 📈 كيف أراقب استهلاك الرصيد؟

### 1. من سجلات الخادم (logs)

**الآن تظهر التكلفة في كل طلب:**
```
✅ Response generated - Tokens: 687 (in:512, out:175) - Cost: $0.0005
```

**التفسير:**
- **Tokens:** إجمالي التوكنز
- **in:** توكنز الإدخال (السؤال + الكونتكست)
- **out:** توكنز الإخراج (الجواب)
- **Cost:** التكلفة بالدولار

---

### 2. من لوحة تحكم OpenAI

**الرابط:** https://platform.openai.com/usage

**ماذا أراقب:**
- **Daily usage** - الاستهلاك اليومي
- **Cost per day** - التكلفة اليومية
- **Remaining balance** - الرصيد المتبقي

---

### 3. تنبيهات الرصيد المنخفض

**إعداد التنبيهات:**
1. اذهب إلى: https://platform.openai.com/account/billing/limits
2. ضع حد أقصى شهري (مثلاً $4)
3. فعّل التنبيه عند 80% استهلاك

## 🚨 ماذا أفعل إذا بدأ الرصيد ينفد بسرعة؟

### الحلول السريعة:

```bash

```

#### 2. تقليل عدد التحاليل في الكونتكست
```python

```

#### 3. إيقاف الخادم مؤقتاً
```bash

```

```python

limiter = Limiter(key_func=get_remote_address)

@router.post("/chat")
@limiter.limit("10/minute")  # 10 طلبات فقط كل دقيقة
async def chat_endpoint(...):
    ...
```

## 📞 الأسئلة الشائعة

### س: متى يجب أن أشحن رصيد جديد؟
**ج:** عندما يصل الرصيد إلى $1 أو أقل

### س: هل يمكن استخدام النموذج المجاني؟
**ج:** لا، OpenAI لا يوفر نماذج مجانية لـ ChatGPT API

### س: كيف أعرف كم طلب أرسلته؟
**ج:** من لوحة تحكم OpenAI → Usage → Total requests

### س: هل التحسينات تؤثر على جودة الإجابات؟
**ج:** قليلاً، لكن الإجابات لا تزال جيدة جداً
